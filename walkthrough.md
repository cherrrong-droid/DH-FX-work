# DH FX work project

이 프로젝트는 금융 스왑 자동화 및 컨펌 생성을 위한 통합 시스템입니다. 각 기능별로 파일이 나뉘어 체계적으로 관리됩니다.

### 1. triggers.gs (통합 컨트롤러)
- 모든 시트의 변화를 감지하고 적절한 자동화 기능을 호출하는 마스터 트리거 파일입니다.
- **`onEdit`**: Command Sheet(A열) 및 날짜 계산기 시트의 주요 셀 입력을 감지합니다.

### 2. myFunction.gs (커맨드 시트 로직)
- 기존의 Command Sheet 처리 로직을 담고 있습니다. 
- 사용자님이 요청하신 대로 원본 로직을 그대로 보존하여 `triggers.gs`에서 안전하게 호출합니다.

### 3. 날짜 계산기.gs (날짜 자동 설정)
- **ODD DATE 처리**: `7/27 ~ 7/28 [1d]` 처럼 사이 공백이 있는 날짜 범위 입력도 정확히 파싱하여 처리합니다.
- `F4`, `F14` 칸 입력 시 하단의 날짜 셀(`F7:G7`, `F17:G17`)을 자동으로 채워줍니다.

### 4. 스프레드 스타팅.gs (환율 계산)
- `C26` 또는 `D26`의 Starting Rate를 기준으로 하단 테너들의 계산값을 자동으로 생성합니다.
- 새로운 시트 레이아웃(`B27` 입력, `C27` 출력)에 최적화되어 있습니다.

### 5. 스프레드 증감.gs (자동 합산/차감)
- **`H26`**: `F26 + G26` 값이 자동으로 계산됩니다.
- **`H27`**: `F27 - G27` 값이 자동으로 계산됩니다.

### 6. 컨포 생성.gs (컨펌 자동 생성)
- **다양한 테너 지원**: 일반 테너뿐만 아니라 `3*6`, `3x6` 같은 조합 테너를 지원하여 자동으로 만기 날짜를 찾아줍니다.
- **ODD DATE 컨펌**: 입력 명령문 자체에 날짜 범위가 있는 경우(예: `7/27~7/28`) 이를 인식하여 정확한 `VAL` 날짜 메시지를 생성합니다.
